# Getting Started

<br>


## Initial Setup
### Backend:
1. **Create virtualenv**:
    ```cmd
    cd .\Backend
    ```
    ```cmd
    python -m venv venv
    ```
2. Install all needed requirements
    ```cmd
    pip install -r requirements.txt
    ```
3. **Run the server**:
    ```cmd
    python manage.py runserver
    ```
### Frontend:
1. **Open new terminal**:
    ```cmd
    cd .\Frontend\my-app
    ```
2. Install all needed requirements
    ```cmd
    npm i
    ```
    
2. **Run the server**:
    ```cmd
    npm start
    ```
